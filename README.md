Sonicware Lofi-12 XT Tool v1.0
A high-speed batch audio converter specifically designed for hardware samplers like the Sonicware Lofi-12 XT. Powered by the legendary SoX (Sound eXchange) engine for maximum precision and performance.

🚀 Features
Batch Processing: Drag and drop multiple files or entire folders.

Hardware Optimized: Default settings tailored for Lofi-12 XT (48kHz, 24-bit, Mono).

Smart Filenames: Automatically replaces spaces with underscores to ensure compatibility with hardware file systems.

Audio Normalization: Peak normalization to 0dB ensures consistent volume across all your samples.

Blazing Fast: Uses the SoX engine to process files significantly faster than general-purpose converters.

Multilingual: Full support for English, Danish, German, Italian, Spanish, and French.

Open Source: Free to use, modify, and distribute.

🛠 Installation
Download the Tool: Download the latest release .exe from this repository.

SoX Engine: This tool requires sox.exe.

You can install SoX on your system.

Alternatively, simply place sox.exe and its associated .dll files in the same folder as this tool.

Run: Open Sonicware Lofi-12 XT Tool.exe and you are ready to go.

📖 How to Use
Add Files: Drag and drop WAV files into the list or use the "Add Files/Folder" buttons.

Set Output: Choose where you want your converted samples to be saved.

Adjust Settings: Select your desired Sample Rate, Bit-depth, and Channels. Enable "Normalize" if you want full-volume samples.

Convert: Hit START CONVERSION. The tool will process the files and automatically open the output folder when finished.
